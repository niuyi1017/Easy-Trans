# Easy-Trans
一款可以翻译多国语言的轻量级webapp —— 简洁、方便、易用    
适合高中生，四六级，考研党  


## 产品简介&特色  
* 极简风格 -- 只展示对你有用的  
* 权威翻译 -- 数据来源于有道词典官方api  
* 自动识别 -- 常见语言自动识别，自动翻译  
* 动态展示 -- 即输即译，无需多余点击
* 轻量级 -- 拒绝鸡肋功能，无广告，无推送，无推荐，专注翻译
